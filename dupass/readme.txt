
This new script allows you to password protect the upload process. This is to ensure that spammers can't upload files to your account.

You can download the older script here : http://www.ampercent.com/downloads/du.rar


1. Open Index.php in an HTML editor
2. Go to line 27 and enter a password. This need not be same with your Dropbox account's password.
3. Go to line 52 and enter your Dropbox account credentials.
4. Go to line 75 and enter the name of the Dropbox folder in value field
5. Upload all files in a directory of your web server.
6. Done !

Pass the address of the page and the password to your friends and let them upload files to your Dropbox account.
